◆ hash_table_load()

HashTable_LoadError hash_table_load ( const char *  filename,
HashTable **  out_table 
)

Loads a hash table from a specified file.

Instead of returning the table, it returns an error code. The created hash table is returned via the out_table pointer. On error, out_table is set to nullptr (or an intermediate state).

Parameters
filenameThe path to the file to load.
out_tableA pointer to a (HashTable*) variable. On success, this will be set to point to the newly loaded table. On failure, it will be set to nullptr.
Returns
A HashTable_LoadError code indicating success (HT_LOAD_OK) or the type of failure.
